I fill the viewport!

Setting height: 100%; on the html and body tags, then setting min-height: 100%; on the main wrapper of your content.

Note: We needed to include the reset.css file for this to work.

CSS Styles

If using React or Next.js, we need to include selectors for every element that sits between the root html tag, and the element we want to stretch to 100% of the screen size. 

          
        html,
        body,
        #root, /* for create-react-app */
        #__next /* for Next.js */ {
          height: 100%;
        }
