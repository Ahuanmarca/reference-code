**Section 33: React Hooks Project**

### **Create React App and install Material UI**  
```
npx create-react-app todo-app-hooks
npm install @mui/material @emotion/react @emotion/styled
npm install @mui/icons-material
```

### **Custom components**

- `TodoForm`
- `TodoList`
- `TodoItem`
